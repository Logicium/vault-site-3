# Vault — Photo Guide

Vault is the **Shop** archetype in the Apotome website program.

Drop final, web-optimized JPGs into `public/photos/` using the **exact filenames** below.

## Variants

| Variant      | Total photos | Gallery shots |
| ------------ | ------------ | ------------- |
| Essentials   | ~10          | 6             |
| Portfolio    | ~18          | 12            |

## Required (Essentials)

| Filename            | Shot                                            |
| ------------------- | ----------------------------------------------- |
| `hero.jpg`          | Hero flat-lay or signature product on color     |
| `about-shop.jpg`    | Wide interior of the shop                       |
| `storefront.jpg`    | Storefront in daylight, sign visible            |
| `product-01.jpg`    | Hero product, square, on neutral ground         |
| `product-02.jpg`    | Second product, hero shot                       |
| `product-03.jpg`    | Third product, hero shot                        |
| `product-04.jpg`    | Fourth product, hero shot                       |
| `interior-01.jpg`   | Shelving / merchandising detail                 |
| `interior-02.jpg`   | Counter / register area                         |
| `lifestyle-01.jpg`  | Goods being used in real life                   |

## Photo specs

- 2400 px on the long edge minimum
- Color-graded JPG, sRGB, quality ~80
- **Square** for product hero shots
- **4:5 portrait** for category covers
- Clean, neutral background for hero products; live in their habitat for lifestyle

## Tips

- Use **one consistent background color** for product heroes
- Shoot lifestyle outdoors at the **golden hour**
- Capture **textures** — knit, leather, ceramic, paper
- Keep brand identity tight: choose a 2-color story per shoot
